# Chartogne-Taillet — recipe (visual)

A grower-champagne house whose entire interface is an **antique engraved terroir map**. You move across
the etching from parcel to parcel; the map IS the navigation. Quiet, expensive, editorial.

## Palette (approx)
- **Parchment** `~#f1ead8` paper ground (warm cream).
- **Engraving ink** `~#1c1a17` charcoal for the etched map + display serif; muted greys for depth.
- **ONE brick-red** `~#a23a36` — highlights the active vineyard parcel and the active index item. The only chroma.

## Type
- **High-contrast Didone / old-style SERIF**, **widely letter-spaced caps** for place names (`LES ORISEAUX`, `LES HEURTES BISES`, `MERFY ET SAINT THIERRY`) — the signature voice.
- Smaller serif for the parcel index; a **♩ musical-note** glyph as a recurring motif (sound).

## Grid & composition
- **Full-bleed antique engraved aerial map** as the hero; centered tracked-serif place-name overlaid.
- **Bottom horizontal PARCEL INDEX** acts as nav/timeline (`Les Barres … Les Heurtes Bises → Avize`), active item in red.
- A second view frames the map with **survey grid coordinates** (A–D rows, 0–7 cols). `MENU` top-left, `MAP` top-right.

## Motion (inferred)
- Smooth **pan / zoom across the engraving** between parcels, slow refined crossfades, gentle parallax on map layers; unhurried luxury pacing (GSAP/Lenis feel).

## Mood
Heritage, terroir storytelling, understated luxury — a centuries-old map you glide through.

## When to use
Heritage / luxury brands (wine, spirits, estates, fashion-heritage); **map-as-navigation** storytelling; editorial-serif elegance with a single restrained accent on cream.
