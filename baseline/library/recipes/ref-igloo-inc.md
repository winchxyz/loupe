# Igloo Inc — recipe (visual)

A consumer-brands holding company (Pudgy Penguins' parent) presented as a **cold cinematic 3D lab**: an
igloo assembles/explodes, and each portfolio company is a **floating ice specimen** examined through a
technical HUD. Premium, icy, instrument-like.

## Palette (approx)
- **Monochrome cold grey-blue** throughout: pale silver-grey fog grounds `~#c2c8ce → ~#aab2ba`, slate-grey objects `~#8a929a`, deep cool shadow `~#5a6168`.
- **Glowing cool-white** `~#ffffff` for the lit brick seams / edges (the only "light").
- White mono text. **No warm hue at all** — desaturation IS the brand.

## Type
- **Monospace for EVERYTHING** — HUD labels, the right-aligned **Manifesto**, `// Copyright © 2026`, portfolio tags (`PORTFOLIO_CO_01 / PUDGY PENGUINS`), instrument readouts (`TEMP 35.26 +01.81`, `D 01.02.2020`), `CLICK TO EXPLORE`, `Sound: On`. Terminal/technical voice.
- **`iGLOO` wordmark** = a chunky rounded-geometric custom logo (slightly retro/pixel) — the one non-mono shape.

## Grid & composition
- **Full-bleed cinematic 3D scene** with a single central hero object that transforms: igloo → exploded floating bricks (glowing seams) → portfolio ice-crystals → a **point-cloud sculpture** on a pedestal.
- **Corner-anchored HUD:** logo top-left, right-aligned Manifesto top-right, `Sound: On` bottom-left, `‹ ›` nav arrows mid-edges to step between specimens.
- **Instrument overlays:** thin leader-lines + constellation data points (numbers like `66 33 57 21 35`), `TEMP`/date/ID readouts, faint wireframe geometry floating in space — the page reads like a scientific scan.

## Motion (inferred)
- WebGL cinematic: igloo **assembles/disassembles** (bricks drift apart), slow camera orbits, ice crystals rotate, **point-cloud morph**, volumetric fog/depth; glowing-seam shader; click-to-explore between portfolio companies; sound-on.

## Mood
Cold-premium, sci-fi research lab — every brand is a "specimen" under glass. Restraint + monochrome + instrument HUD = expensive-feeling.

## When to use
Premium tech / Web3 / holding-co / research brands; a **"specimen examination" portfolio metaphor**; monochrome icy cinematic 3D with mono-terminal HUD overlays (leader-lines, readouts, constellation points). Reach for: total desaturation + one glowing light + all-mono type + a transforming hero object.
