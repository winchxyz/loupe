# Messenger (abeto) — recipe (visual)

A narrative experience rendered as a **cel-shaded (Ghibli/anime) toon 3D world** you walk through as a
child in a red jacket. Intro is a tiny floating island; the body is a hand-painted Japanese street.

## Palette (approx)
- **Teal / seafoam** `~#67b3ad` water + sky base (the unifying ground color).
- Painterly **warm naturals** — building greys, greens, warm tans — all flat-toon shaded with outlines.
- **Red** `~#c0392b` jacket = the moving focal pop; **yellow** `~#e9c44c` accent (BEGIN button, signage).

## Type
- **Chunky BLOCKY display font** with outline / slight 3D extrude — stacked `MES / SEN / GER`, `MAIN SQUARE`, `BEGIN`. Reads pixel-game / picture-book.
- Diegetic CJK signage in the world adds place.

## Grid & composition
- **Intro:** a tiny floating photogrammetry/illustrated **island** (houses, trees, debris) centered on flat teal water with painterly splashes; stacked-block title; chunky **yellow BEGIN** button.
- **Body:** **third-person** cel-shaded street scene — avatar from behind walking a Japanese alley; corner UI icon stack (list · music · wardrobe); location label (`MAIN SQUARE`) bottom-left in the blocky font.

## Motion (inferred)
- **Walk a toon-rendered 3D world** (third-person follow cam). **Cel/outline shader** + painterly cloud & water; ambient sound; chapter/location reveals. Exploration-narrative, not scroll.

## When to use
