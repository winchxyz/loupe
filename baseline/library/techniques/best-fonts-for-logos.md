# Logo & wordmark typeface selection — applied rules

Core idea: a logo says a lot with very little, and the font does most of the talking. Pick the typeface from the brand's TONE first, then verify it survives at small sizes. Almost all of these are free Google Fonts — reach for a named one below instead of defaulting to the system sans.

## Pick by tone (named fonts, when to reach for each)

### Modern / tech / startup (geometric sans)
- **Futura / Avenir / Proxima Nova** — geometric, Bauhaus-clean, forward-thinking. Use for tech, architecture, design firms wanting "simple + modern." (Avenir = French for "future.") Proxima Nova is the digital-first workhorse (renders well across all media).
- **Poppins / Quicksand** — rounded geometric, friendly and approachable; rounded terminals soften the tech feel. Use for startups/retail/wellness that want modern but warm.
- **Plus Jakarta Sans / Instrument Sans / Rethink Sans / Exo 2** — current geometric sans with a little personality (Exo 2 is sharper/more futuristic for gaming/tech).
- **Lato / Work Sans / Montserrat** — neutral, highly legible, versatile defaults; safe when in doubt. Montserrat carries a faint urban/historical charm at any size.
- **Space Mono** — monospaced, technical/retro-1960s; use ONLY when you want an overtly dev/code/futuristic signal.

### Luxury / editorial / high-end (high-contrast serif)
- **Didot / Bodoni** — dramatic thick/thin contrast, vertical stress, commanding. The go-to for fashion, cosmetics, premium publications. Needs size — thins disappear when small.
- **Playfair Display / Abril Fatface / Gilda Display / Prata** — high-contrast display serifs for premium product, beauty, high-end restaurants. Abril Fatface = boldest statement.
- **Garamond / Cinzel** — classic, restrained, refined (Cinzel = Roman-inscription caps, regal). Use for "quality and refinement" without drama; hotels, fine dining.
- **Instrument Serif / Alegreya / Taviraj / Rufina** — refined but approachable serifs; sophistication without being cold or overly formal.

### Friendly / playful / expressive (display + script)
- **Sniglet / Bree Serif** — rounded, bouncy; kid-focused, food, family-friendly, community.
- **Shrikhand** — thick rounded retro display (Indian street-lettering); loud, fun, vibrant packaging/restaurants.
- **Alfa Slab One** — heavy slab serif that demands attention; sports, entertainment, max visual presence.
- **Grand Hotel / Quintessential** — scripts with vintage/handcrafted charm; cafes, bakeries, weddings, artisanal. Keep scripts large enough to stay readable.

### Accessibility-first
- **Inclusive Sans** — open shapes, generous spacing, designed for readability; reach for it when clarity/inclusivity is the brief (education, healthcare, nonprofit).

## Selection rules (do / choose / avoid)
1. Match the typeface to brand identity first — values, voice, market positioning drive the choice, not personal taste.
2. Legibility is the top constraint — test the wordmark at favicon/nav size AND hero size before committing; if the thins vanish small, it fails.
3. Choose for the audience — pick what resonates with the target consumer, not what the designer likes.
4. Demand adaptability — needs enough weights/styles for every surface (digital + print). Prefer **variable fonts** for finer styling control.
5. Know the four families and what they signal: **display** = impact, breaks at small size; **sans serif** = modern/minimal/tech; **serif** = tradition/authority/elegance; **script** = personality/flair, watch readability.

## Pairing rules (when a logo needs two fonts)
- You usually don't need a second font; add one only for a tagline/sub-lockup. Goal: contrast WITHOUT clashing — bold headline font + minimal companion = clean hierarchy.
- Proven pairs from the source:
  - **Montserrat + Hind** — modern, approachable
  - **Futura + Lora** — sleek, sophisticated
  - **Poppins + Libre Baskerville** — contemporary yet trustworthy
  - **Grand Hotel + Oswald** — retro, bold
- Pattern: pair a geometric/display lead with a quieter neutral (sans+humanist-sans, or geometric-sans+serif) so the two roles read as distinct.

## Build checklist
- [ ] Font chosen from brand TONE (not the default system sans), using the tone buckets above?
- [ ] Wordmark legible at nav/favicon size as well as hero size (high-contrast serif thins survive)?
- [ ] Family signal matches intent (sans=modern, serif=premium, script=expressive, slab/display=loud)?
- [ ] Enough weights/styles (or a variable font) for every surface, digital and print?
- [ ] If a second font is used, it contrasts with the lead without clashing (one clear hierarchy)?
- [ ] Script/display picks confirmed readable at their smallest real placement?

> Ties to our experience: maps to our TESQ wordmark work — the SQUARE filled Q kept in the same Inter Tight weight as TES is exactly this "contrast without clashing / one consistent family signal" rule, and the wireframe tesseract-Q was rejected for failing legibility at small size.
