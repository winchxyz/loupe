# Brochure layout patterns — applied rules

Core idea: a brochure is a *guided sequence* of panels, not a poster. Pick the FOLD/FORMAT first (it dictates how many discrete panels you get and the reading order), then map one job to each panel, and end on a single call to action. Format is the structural decision; everything else hangs off it.

## Format → use-when (choose the container first)
1. **Tri-fold** (paper into thirds) — choose for a 6-panel pitch with a fixed sequence: cover hook → intro/about → numbered services → proof → contact/CTA. Reach for it for agencies, resumes, quick promos, advocacy. Most common; the folds force a natural narrative order, so design *to* the panel breaks, never across them.
2. **Bi-fold** (folded in half) — choose when you have more copy than a tri-fold panel holds but don't need a booklet. Good for A4/A5 business profiles where each spread carries one substantial section.
3. **Z-fold** — like tri-fold but accordion ("Z") so panels reveal in strict left-to-right order. Choose when reveal order must be enforced (a step-by-step or unfolding story).
4. **Bound / booklet** — choose for high page counts (8, 18, 24, 26, 30+ pages): catalogs, company profiles, magazines, white papers, wedding/lookbook. Use a repeating master spread so 20+ pages stay visually cohesive; vary accent color per section, not the grid.
5. **Flat / one-page** (single or two-sided) — choose for a quick takeaway: menu insert, IT one-sheet, service+price card, direct-mail flyer. One side = hook, other side = detail. Laminate-friendly reference, not a story.

## Reusable panel/content patterns (extract, then reuse)
6. **Numbered feature list** — present services/benefits as an explicitly numbered set. Use when you have 3-6 discrete offerings; numbering creates scan-order and implies completeness. (Recurs across software, purple-bubble, A4 profile templates.)
7. **Cover = single image + logo, nothing else** — keep the front minimal: one strong photo, the logo, maybe one line. Use for fashion/art/minimal brands where the image is the message; resist filling the cover.
8. **"About Us" with headshot placeholders** — reserve a panel/back-page with portrait slots + names for team/people-driven trust. Use for agencies, small teams, consulting.
9. **Two-column package + price** — one column lists what's included, the adjacent column lists cost/extras. Use for service businesses (weddings, IT, travel packages) where the buyer is comparing scope vs. price.
10. **Two-page product unit** — for catalogs, give each product a spread: page 1 = lifestyle/detail, page 2 = specs/nutrition/facts. Use for product lineups so every item gets identical, comparable real estate.
11. **Physical→digital bridge (QR / contact block)** — place a QR code and full contact line (phone, email, website, address) on a dedicated panel. Use on any print piece that must convert offline → online; never bury contact in body copy.
12. **Full-page pull-quote / case-study break** — interrupt dense reading (white papers, long profiles) with full-page quotes and dedicated case-study pages. Use to re-engage the reader and pace a long document.

## Content discipline (the article's own rules, as decisions)
13. **One central message per piece** — do *not* overload; cutting secondary info so the key idea survives beats cramming. If a detail buries the headline, drop it.
14. **Always end on one CTA** — every brochure must tell the reader the next action (visit site / call / come in). One CTA, unmissable.
15. **Keep copy concise** — respect attention; tight copy holds it. Trim every line that isn't load-bearing.
16. **Match tone to audience** — a professional/B2B reader gets a restrained palette; playful/colorful is wrong there. Decide audience first, then aesthetic.
17. **Color carries meaning — choose deliberately** — blue = trust/serenity/calm (health, finance, institutions); soft/warm tones = romance/refinement (wedding, wine); bright/vivid = energy/attention (food, education, kids). Pick the scheme for the *connotation*, not just taste, and keep one accent consistent across panels.

## Size cheat-sheet (pick for carry vs. content vs. cost)
- **5.5"×8.5" Half-letter** — compact handout/insert, quick takeaway, most likely to be picked up at an event.
- **8.5"×11" Letter** — default; best balance of size vs. information.
- **8.5"×14" Legal** — more room for detailed info + imagery.
- **11"×17" Tabloid** — largest/most eye-catching; highest material + print cost. (Bigger = costlier; only size up when the content or impact earns it.)

## Build checklist (tick while applying)
- [ ] Format/fold chosen FIRST, and the layout respects panel breaks (nothing important straddles a fold)?
- [ ] Exactly one central message; secondary info cut rather than crammed?
- [ ] One unmissable CTA + a contact/QR block present?
- [ ] One accent color held consistent across all panels/spreads, chosen for its connotation to the audience?
- [ ] For bound pieces: a repeating master grid so many pages read as one system?
- [ ] Copy trimmed to essentials; tone matches the target reader (restrained vs. playful)?
