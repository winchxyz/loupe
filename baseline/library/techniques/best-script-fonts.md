# Script fonts — named picks + deployment rules

Core rule: a script face is a SEASONING, not a body text. Use it for ONE thing per view (logo, hero word, or short heading), pair it with a plain neutral, and verify legibility at the real render size before you commit. One flourish too many and the design reads "off."

## Pick by job (named, free Google Fonts)
Reach for the face whose temperament matches the brief — don't pick on "prettiness" alone.

1. **Formal / elegant / wedding-tier** — choose `Great Vibes`, `Alex Brush`, `Allura`, `Pinyon Script`, `Petit Formal Script`, or `Parisienne`. High-contrast calligraphic strokes; read as romance/luxury. Allura, Beau Rivage, and Petit Formal Script hold up smallest, so prefer them when the script must shrink.
2. **Luxury / beauty / fashion brand mark** — choose `Luxurious Script`, `Great Vibes`, `Style Script`, or `Euphoria Script`. Smooth swelling strokes + small swirl details = upscale. Pair with a thin/light sans for the wordmark lockup.
3. **Casual / friendly / playful** — choose `Dancing Script`, `Caveat`, `Pacifico`, `Yellowtail`, or `Grape Nuts`. Bouncy baselines and even strokes; warm and approachable. Caveat is the most readable of the casual set — default to it when the script carries real words.
4. **Kids / educational / handwritten** — choose `League Script`, `Caveat`, `Meow Script`, or `Grape Nuts`. Open, even, forgiving letterforms.
5. **Bold display / attention-grabbing headline** — choose `Marck Script`, `Rouge Script`, `Passions Conflict`, `Molle`, or `Neonderthaw`. Thick strokes / extreme slant / exaggerated ascenders+descenders. These trade legibility for impact — keep them to a few large words.
6. **Retro / vintage / signage** — choose `Tangerine` (1970s), `Pacifico`, `Neonderthaw` (neon-tube), or `Rouge Script`. For `Neonderthaw` specifically, set it in ALL CAPS on short strings to stay readable.
7. **Monoline / clean branding & packaging** — choose `Sacramento` (even stroke width, multi-language) or `Playwrite USA Traditional` (consistent-slant cursive, 150+ Latin languages, multiple weights). Best when you want script warmth without high-contrast fragility.

## Deployment decisions
- **One script per layout.** Multiple decorative scripts = clutter. Limit it to the logo OR the hero word OR a section heading — not all three.
- **Always pair with a plain neutral.** Set body and UI in a simple sans or serif; never set paragraphs, nav, or form text in a script. The contrast is what makes the script read as special.
- **Match temperament to audience** (do this before picking on looks): playful script for a young audience, calligraphic/formal for luxury and events. A mismatch reads wrong even if the face is beautiful.
- **Tune kerning AND tracking by hand.** Scripts ship with connecting strokes that assume tight spacing — too tight reads cramped, too loose breaks the connected flow and looks disjointed. Adjust letter-spacing per lockup, don't trust defaults.
- **Avoid over-stylized faces when the text must be read.** If a string longer than a few words has to be legible, drop the extreme-slant/extreme-flourish picks (Molle, Passions Conflict, Neonderthaw) for a clearer one (Caveat, Sacramento, Rochester, Petit Formal Script).
- **Verify at the real size, on the real background.** Test the chosen face at the size it actually renders, over the actual background, with enough contrast for the strokes to separate. A thin high-contrast script can vanish on a busy or low-contrast hero.

## Build checklist
- [ ] Exactly one script face in this view (logo / hero / heading — pick one).
- [ ] Body, nav, and form text set in a plain sans or serif, not the script.
- [ ] Face temperament matches the audience (playful vs. calligraphic), not just "looks nice."
- [ ] Kerning/tracking hand-tuned — not cramped, not so loose the connections break.
- [ ] Legibility confirmed at the actual render size, over the actual background, with adequate contrast.
- [ ] If the string is long or small, swapped any extreme-slant/extreme-flourish face for a clearer one.
- [ ] `Neonderthaw` (or similar neon/extreme face) set ALL CAPS on a short string.
