# Website color schemes — applied rules

Color reads before content: the palette sets the mood and the visual hierarchy before a single word lands. The article is a 53-swatch roundup, but the reusable signal is (a) which mood family fits the brief, (b) how to use one accent against a quiet base, and (c) the palette-discipline + accessibility rules underneath all of it. Skip the brand names; keep the patterns.

## Step 1 — pick a mood family from the brief (not from taste)
Each family maps to an industry/emotion. Choose by what the site must communicate.
1. **Minimal / neutral** (whites, blacks, grays, soft pastels) — elegance, focus, readability. Default for portfolios, luxury/fashion, wellness, anything content-first. Quiet base; let one accent or the content carry the energy.
2. **Warm** (red / orange / yellow, earthy browns) — energy, comfort, appetite, nostalgia. Food, home goods, hospitality, entertainment, creative/individual brands.
3. **Cool** (slate grays, crisp blues, teal) — calm, intelligence, trust, security. Finance, tech, healthcare, law — anything that must read as credible and dependable.
4. **Vibrant / bold** (saturated, high-contrast) — attention, fun, excitement. Fashion, gaming, kids, music — used sparingly so it energizes instead of shouting.
5. **Modern** (limited palette, muted tones, one pop) — innovation, sophistication, minimalism. SaaS, tech, software. Lean on clean type + restraint, not many colors.

## Build decisions (each = a do / choose / avoid)
6. **Cap the palette at 3–5 roles** — one dominant, one or two secondary, one accent. More colors dilute the brand and read as chaotic. Avoid "rainbow" decks.
7. **Quiet base + one loud accent** — keep most of the page neutral/muted and reserve a single vivid color (cinnamon, cinnabar, coral, mint, teal) for CTAs, links, and highlights. The accent is the star precisely because it's rare; spend it on action, not decoration.
8. **Use the accent to build hierarchy, not just to color** — the eye goes to the highest-contrast spot, so the most saturated/contrasting color should sit on the thing you want clicked (the primary CTA). Don't waste max contrast on a decorative block.
9. **Earn depth with dark anchors** — within a soft or pastel scheme, add one deep tone (chocolate brown, navy, maroon, blue-gray, pewter) to ground the design and make lighter elements pop. Avoid an all-light palette with no anchor — it reads flat and weak.
10. **Reach for complementary pairs when you need a strong call-out** — opposite hues (purple + yellow, blue + orange) give the sharpest "look here" contrast for headlines and CTAs. Use the pair sparingly; a whole page of complementaries fights itself.
11. **Monochromatic = one hue, many values** — gray-on-gray ink-wash, or maroon→blush, reads refined and unified. Use it for luxury/editorial/galleries; lean on value (light/dark) for hierarchy since you have no second hue to contrast with.
12. **Apply color psychology deliberately** — red = energy/urgency, blue = trust/calm, green = nature/growth, yellow = optimism. Pick the dominant hue to reinforce the message, not to decorate against it.
13. **Map mood to value temperature** — bright + high-contrast = playful/energetic; muted + earthy = warm/nostalgic; dark grays/navy/black = moody, edgy, authoritative, premium. Match the value scheme to the feeling the brief names.

## Accessibility & verification (non-negotiable)
14. **Contrast text vs. background, always** — light bg + dark text or vice versa. Check with a contrast checker against WCAG; never ship a pastel-on-pastel or low-contrast accent-on-accent block for body copy. Color-blind and low-vision users are the gate.
15. **Test the palette across devices and lighting** — the same hex looks vibrant on one screen and muted on another (and shifts with brightness/HDR). Preview phone + tablet + desktop, light and dark ambient, before calling it done. A swatch that's only verified in one viewing condition is unverified.

## Build checklist (tick while applying)
- [ ] Mood family chosen from the brief's industry/emotion, not improvised?
- [ ] Palette is 3–5 roles (dominant / secondary / accent), not more?
- [ ] Exactly one accent reserved for CTAs/links/highlights, against a quiet base?
- [ ] At least one dark anchor grounds any soft/pastel scheme?
- [ ] Primary CTA sits on the highest-contrast / complementary pairing?
- [ ] Text/background contrast passes a WCAG contrast checker?
- [ ] Palette previewed on phone + desktop, light and dark conditions?
