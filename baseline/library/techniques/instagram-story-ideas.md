# Vertical story/poster layouts — applied rules

Core idea: a story frame has ~2 seconds and one tap to land its message. Treat each as a single-screen poster with exactly one job: one focal point, one CTA, one readable headline, everything else subordinate. The 21 templates collapse into a handful of reusable composition patterns — steal the structure, not the brand.

## Reusable composition patterns (each = a layout you can drop in)
1. **Center-stage subject** — one hero photo/product dead center, name big above or on it, fine print small below. Use for product launches, spotlights, book/movie picks, cross-post screenshots. Choose when the asset itself is the message; give it a transparent-background cutout so it reads instantly.
2. **Top-to-bottom gaze path** — title at top → details in the middle → decorative image/CTA at the bottom. Use for webinars, events, talks; it walks the eye through who/what/when/where in reading order. Reinforce the path with a zig-zag or diagonal so the gaze keeps moving down to the CTA.
3. **Collage / layered photo stack** — overlapping rectangles or a multi-cell grid filled with shots from one shoot. Use for fashion drops, lookbooks, speaker line-ups; adds movement and dimension. Keep ALL crops from the same shoot so the set reads as one aesthetic, not a clip-art pile.
4. **Bordered-frame card** — a textured/colored border wrapping a central content block (price tag, envelope, notepad, ticket motif). Use for sales/promo codes, save-the-dates, invitations; the frame metaphor does the explaining. A circular cutout + paper texture = "price tag" without a single word of UI chrome.
5. **Type-as-hero (minimal)** — oversized condensed/display word fills the frame, near-zero decoration (just a hairline rule top/bottom). Use for promotions, milestones, NYE/maximum-impact moments. Abbreviate so the word never has to shrink to fit; let scale + contrast carry it.
6. **Playful-background, bold-foreground** — bright/curved shapes or floral/semicircle motifs behind a single bold CTA word. Use for AMAs, hiring, workshops; keeps it warm and inviting while the call to action stays unmistakable. Decoration goes BEHIND and never competes with the headline.

## Build decisions distilled from the templates
- **One display + one workhorse font, paired for contrast.** Pair a characterful display/headline face with a clean legible body (Poppins+Gilda Display, Unbounded+Rethink Sans, Overpass Mono+Xanh Mono). Headline carries personality; body stays neutral and readable.
- **Match font personality to the message:** handwritten/script = warm & casual (Nothing You Could Do); rounded geometric = friendly-clear (Plus Jakarta Sans, Poppins); condensed all-caps = urgent CTA (Instrument Sans, Anton); mono = high-tech/futuristic (DM Mono); warpy display = loud & attention-grabbing (Tilt Warp, Gasoek One).
- **Guarantee headline-vs-background contrast** — high contrast is called out repeatedly as what keeps text legible over busy/photo backgrounds. If a photo sits behind type, add an opacity layer or overlay so the photo blends and the words still pass.
- **Recolor, don't redraw.** The whole template system theming is "swap the background/accent to the brand or occasion color" — build with one accent variable so the same layout reskins (pink brunch / navy dinner / gold glam) without touching structure.
- **Put the action one tap away.** Every conversion-oriented frame uses a Link/Link-sticker straight to the destination — "don't make them hunt." In web terms: the CTA is a real link/button at the focal point, not buried.
- **Bake interactivity in** where the medium allows — polls, questions, emoji sliders, "Add Yours" make a static frame two-way; reach for an interactive element instead of one more decorative shape.

## Build checklist (tick while composing a story/poster/hero panel)
- [ ] Exactly one focal point and one CTA per frame — is anything else competing for the eye?
- [ ] Headline readable at a glance: big, high-contrast, abbreviated enough that it never shrinks to fit?
- [ ] Two-font system — one display for personality, one neutral workhorse for details?
- [ ] Font personality actually matches the message (warm vs urgent vs high-tech)?
- [ ] Accent/background driven by a single variable so it reskins per brand/occasion?
- [ ] CTA is a live link/button at the focal point, one tap from the destination?
- [ ] If a photo sits under text, an overlay/opacity layer guarantees legibility?
