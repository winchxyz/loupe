# Flyer / promo-poster layout patterns — applied rules

Core idea: a flyer is a single-glance unit. One reader, a few seconds, one decision. Every pattern below trades on the same mechanic: a dominant focal element, one mood-setting style, and the where/when/what readable in under 30 seconds. These translate directly to web heroes, splash screens, promo sections, and event cards — collapse 25 brand examples into the handful of underlying compositions.

## Reusable patterns (extract the composition, not the brand)
1. **Bold-type + power-image impact** (Skate Club, Running Club, New Year's, Hack Day). One oversized headline, one dominant energetic image/illustration, high contrast. Use when the goal is instant impact and energy — sports, events, countdowns, launches. Build: 1 hero word at display scale, visuals push the eye toward the headline, not away from it.
2. **Minimalist + negative-space premium** (Gala, Open House, Furniture Studio). Generous whitespace, refined type, restrained palette, one tasteful image. Use for upscale/luxury/real-estate/high-ticket — the empty space IS the signal of premium. Build: give the focal element room; resist filling the canvas; let date/price/dress-code breathe.
3. **Elegant-traditional / editorial** (Classical Concert, Cocktail Party). Refined serif-leaning typography, muted/sepia or tonal palette, clear hierarchy. Use for cultural, formal, sophisticated, "high-brow" moods. Build: tone the palette down, lean on type weight contrast not color for hierarchy.
4. **Warm / friendly / approachable** (Food Drive, Volunteer, Cafe Opening, Dog Walker). Soft palette, gentle imagery, friendly fonts, human/photo focus. Use when the ask is participation, donation, trust, or community — frame the CTA as opportunity, not obligation. Build: real human photos over stock illustration; soft colors; warm copy.
5. **Playful / cheerful** (Yard Sale, Floral Invite, Spring Art Sale). Bright colors, cheerful graphics, lighthearted fonts. Use for casual, personal, seasonal, low-stakes events. Build: bump saturation, add decorative graphic elements, keep layout simple so what/where/when still read at a glance.
6. **Themed-nostalgia / retro** (Retro Record Sale, Pixel Hack Day). A committed period aesthetic — distressed textures + vintage type, or pixel art + blocky font. Use when standing out from generic promos matters and the audience reads the reference. Build: commit fully to ONE era; the aesthetic is the differentiator, but key details (dates, prices) must still survive the styling.
7. **High-impact pattern + contrast** (Bold Patterned Retail, Modern Retail Sale). Vibrant repeating pattern or color block + striking type + strong contrast; sale % made unmissable. Use for urgency-driven retail, flash sales, limited-time offers. Build: make the number/discount the largest readable element; pattern energizes but never competes with the offer.
8. **Structured / sectioned information** (Workshop Session, Roommate Search, School Orientation, Music Lessons). Distinct labeled blocks for dense info (rent/amenities/contact, schedule/map, pricing/bio) so nothing feels cluttered. Use when there's genuinely a lot to communicate. Build: separate description from logistics with type weight; one section = one question answered.
9. **Botanical / organic** (Floral Invite, Botanical Retail, Furniture Studio). Soft natural palette + botanical/abstract illustration for a fresh, organic, eco/wellness feel. Use for garden, wellness, eco, spring/summer themes. Build: organic illustration as accent framing the composition, not burying the message.

## Cross-cutting build rules (apply to every pattern)
1. **One focal point, decided first.** Title, hero image, or logo — pick what reads first and make size/weight/placement enforce it. Everything else is subordinate.
2. **The 30-second test.** Date, time, location, and CTA must be findable in under 30 seconds. If a viewer has to hunt, the hierarchy failed.
3. **Pair fonts by role.** Bold display font for the headline, clean readable font for body. Two roles, never two competing display faces.
4. **Concise, benefit-led copy.** Active voice, bullets for scanning. The flyer sells the next action, not the full story.
5. **Brand color + seasonal accent.** Stay on brand palette; weave a seasonal accent only as an accent, not a takeover.
6. **One clear, prominent CTA.** Tell people exactly what to do next and make it visually stand out — don't bury it.
7. **Contrast for legibility.** Text must survive its background; never let imagery eat the copy. (Echoes our verification habit — check the rendered contrast, don't trust the comp.)
8. **Whitespace is structure.** Breathing room reads as cleaner and more inviting; crowding reads as cheap.
9. **High-quality, recognizable visuals.** Crisp images that read from a distance; avoid pixelation (unless pixel art is the deliberate theme).

## Build checklist (use when reviewing any promo unit)
- [ ] Exactly ONE focal element wins the first glance (size + weight + placement agree)?
- [ ] Date / time / location / CTA all findable in under 30 seconds?
- [ ] Headline = display font, body = clean readable font — no two display faces fighting?
- [ ] One prominent CTA that visibly stands out and says what to do next?
- [ ] Text passes contrast against its actual rendered background (not just the mockup)?
- [ ] Whitespace gives the focal element room instead of cramming the canvas?
- [ ] The chosen mood pattern (impact / premium / friendly / retro / structured) matches the message — not defaulted?
