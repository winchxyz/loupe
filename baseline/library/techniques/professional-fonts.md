# Professional fonts — named picks by the job they do

Don't pick a font by name recognition; pick by the JOB. First decide the role (UI/body, long-form reading, display/headline, luxury editorial, multilingual), then pull a named option below. All are Google-Fonts-available (free, commercially licensable) unless you confirm otherwise.

## Pick by role (do X / choose Y)

### Body + UI sans (default workhorse — choose one and commit)
Reach here first for app chrome, dashboards, marketing body copy. All are neutral, high-x-height, screen-optimized.
- **Inter / Source Sans Pro / Work Sans** — neutral, built for UI; safest default when in doubt.
- **Plus Jakarta Sans** — geometric with slightly rounded corners → modern + trustworthy + a touch friendlier. Use when "corporate but approachable."
- **Lato** — warm semi-rounded; choose when you want friendly/transparent over strictly neutral.
- **Urbanist / Hind** — geometric, scale cleanly down to small sizes and across devices; good for data-viz labels and mobile.
- **Public Sans / Inclusive Sans** — when accessibility is a hard requirement (open counters, clear letterform distinction). Default to these for gov/edu/a11y-critical work.

### Long-form reading serif (body text that gets READ)
Choose a serif with moderate (not extreme) contrast so it stays comfortable at paragraph length.
- **Lora** — calligraphic roots, moderate contrast; best all-round body serif for blogs/articles/e-books.
- **Source Serif 4** — has optical sizes → use one family for both body and headline and let the optical size do the work.
- **EB Garamond / Crimson Text** — classic old-style warmth; reach for "tradition + authority" (academic, editorial, books).
- **Libre Baskerville** — refined, slightly more formal; sophistication for institutional/academic sites.
- Avoid high-contrast display serifs (Bodoni, Didot) for body — thin strokes break up and tire the eye.

### Display / headline (command attention, set sparingly)
These earn their keep at large sizes only; never set paragraphs in them.
- **Archivo Black / Bebas Neue** — heavy/condensed; space-efficient, industrial, confident. Bebas for tall tight banners.
- **Instrument Serif** — bold geometric serif headline with ambition/innovation energy.
- **Domine** — structured serif that brings authority to headlines while staying contemporary.
- **Aleo** — slab serif; distinctive headline with more personality than a plain sans, still readable.
- **Ramaraja** — high-contrast vintage character; use when the headline itself is the brand statement.

### Luxury / high-end editorial (fashion, premium brands)
- **Bodoni Moda / GFS Didot** — extreme thick/thin contrast, unbracketed/refined serifs → prestige and drama. Display ONLY (headlines, covers). Pair with a quiet neutral sans for body.
- **Raleway** — thin, graceful geometric sans for an elegant luxury feel without a serif.

### Multilingual / international
- **Noto Sans** — broad script/language coverage; default when content spans non-Latin scripts and you need one consistent family.

## Pairing rules (when using two fonts)
- Pair for CONTRAST of role, not similarity: a display serif over a neutral body sans (e.g. Instrument Serif / Bodoni Moda headline + Source Sans Pro or Inter body) is the reliable move.
- Keep body neutral and let the headline carry personality — not both.
- Don't pair two fonts from the same category that are nearly alike (two humanist sans, two old-style serifs) — reads as a mistake, not a choice.
- A single superfamily with optical sizes (Source Serif 4 / Source Sans Pro) is a safe one-font system when you want zero pairing risk.

## Build checklist
- [ ] Chose the font by ROLE (body/UI vs reading vs display vs luxury vs multilingual), not by vibe alone.
- [ ] Body/reading font has moderate contrast and high legibility at its real size; tested at the actual on-screen px, not just in a specimen.
- [ ] Display font used at large sizes only — no paragraphs set in it.
- [ ] If two fonts: they contrast in role; body stays neutral; hierarchy is unambiguous.
- [ ] Verified legibility holds across Web, mobile, and print/responsive breakpoints.
- [ ] Confirmed the license covers commercial use for every weight shipped.
