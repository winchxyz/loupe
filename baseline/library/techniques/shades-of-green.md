# Designing with green shades — applied rules

Core idea: "green" is not one decision. Pick the SUB-FAMILY first (which way it leans — yellow, blue, gray, brown), because the lean carries the meaning; then tune lightness for mood and lock contrast for accessibility. Green sits at ~555nm, the peak of human luminance sensitivity, so it reads easily but also fatigues if oversaturated across large areas — reserve vivid greens for small, intentional marks.

## Pick the sub-family by the message (do X / choose Y)
1. **Want growth / fresh / energetic / "success"** → yellow-greens. They read renewal and happiness. Use for positive-state UI (success toasts, confirmations, "completed").
   - Spring green `#00F0A8` — success / positive actions / confirmations.
   - Lime green `#89F336` — wayfinding cue, a vibrant pop guiding navigation; tiny doses only.
   - Chartreuse `#CCFF00` — near-neon; an attention accent, never a fill or text color.
   - Apple green `#8DB600` — playful accent.
   - Pistachio `#93C572` — the calm one in this family; soft pastel touch, sophisticated next to black/gray.
2. **Want trust / dependable / calm** → blue-greens (ocean register). Use for finance, healthcare, utility, "reliable" brands.
   - Teal green `#006D5B` — strong default for buttons, links, CTAs (deep enough for white text).
   - Sea green `#2E8B57` — relaxed, welcoming environment background/surface accent.
   - Turquoise `#A0D6B4` — light; travel / nature / tech surfaces, not text.
3. **Want serene / sophisticated / understated** → gray-greens. Lower chroma = quieter UI. Use for calm backdrops, overlays, editorial.
   - Sage `#98A869` — calming backdrops, subtle overlays.
   - Gray asparagus `#465945` — dark enough to carry a CTA and hold white text.
   - Seafoam `#93E9BE` — soft surface; pair with a bold pop (coral, sunshine yellow) so it doesn't go flat.
4. **Want premium / natural / formal / earthy** → brown-greens + dark forest greens. Use for luxury, outdoor, heritage.
   - Forest green `#2E6F40` — premium/exclusive accents and CTAs.
   - Olive `#636B2F` — earthy, low-strain; outdoor/adventure, large areas tolerable because muted.
   - Moss `#8A9A5B` — warm golden-undertone natural brand tone.
5. **Tune mood with lightness, not hue.** Lighter tints (add white) → upbeat, dynamic, casual. Deeper shades (add black: forest, hunter, emerald) → dignified, elegant, premium. Decide mood, then shift along the value axis within the chosen family.
6. **Reserve vivid greens for marks, not fields.** Chartreuse/lime/spring belong on CTAs, badges, indicators, wayfinding — small intentional areas. For large backgrounds use the muted/gray/olive end to avoid eye strain.
7. **Use green to signal "go" / positive state, but never as the ONLY signal.** Always carry a second cue (icon, label, position) because red-green colorblind users (deuteranopia) can't separate it from red.

## Accessibility & pairing (non-negotiable)
- **Check contrast, every time.** Light greens (turquoise, seafoam, lime, chartreuse, pistachio) fail on white — don't put text/icons in them; use them as fills behind dark content or as accents. Deep greens (teal `#006D5B`, forest, gray asparagus, the brand darks like `#00674b`/`#004526`) safely hold white text.
- **Never encode meaning in green alone.** Success ≠ "it's green" — pair with a checkmark/label so deuteranopia users aren't lost.
- **Complementary contrast pop:** green's complement is red/coral — a soft green surface (seafoam, sage) wakes up with a coral or sunshine-yellow accent.
- **Cultural check:** green's symbolism shifts by region; confirm meaning for the target audience before committing a green-led brand.

## Brand reference points (deep greens = premium/trust)
- Whole Foods `#00674b`, Lacoste `#004526` — both very dark blue-greens for premium positioning.
- Eco/sustainability brands lean energizing mid-greens (Ecoist Irish green `#0C9A00`).

## Build checklist
- [ ] Chose the sub-family (yellow / blue / gray / brown-green) from the MESSAGE, not by eye?
- [ ] Set mood by lightness — tint for upbeat, deep shade for premium?
- [ ] Vivid greens limited to small marks (CTA/badge/indicator), muted greens for large fields?
- [ ] CTA/text-on-green passes contrast (deep green + white; never text on a pale green)?
- [ ] Any "green = success/go" also has a non-color cue (icon/label) for deuteranopia?
- [ ] Added a complementary pop (coral/yellow) if a soft green surface looks flat?
- [ ] Checked green's cultural meaning for the target region before going green-led?
